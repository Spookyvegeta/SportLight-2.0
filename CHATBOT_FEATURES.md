## 🚀 Technical Details

**API**: Google Gemini Pro
**Rate Limit**: 60 requests/minute (free tier)
**Response Time**: ~1-2 seconds
**Fallback**: Built-in knowledge base
**Security**: API key stored in environment variables

## 💡 Usage Tips

1. **Be specific**: "Tell me about Virat Kohli's batting average" is better than "cricket"
2. **Ask anything**: Don't limit yourself to predefined questions
3. **Follow up**: Ask related questions for deeper understanding
4. **Use quick actions**: Navigate app features instantly
5. **Try different sports**: The bot knows about ALL sports!

## 🔧 Setup Required

To enable Gemini AI:
1. Get free API key from https://makersuite.google.com/app/apikey
2. Add to `.env.local` file
3. Restart frontend server

See `GEMINI_SETUP.md` for detailed instructions.
